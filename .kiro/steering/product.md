# Product Overview

AAE (Automated Ad Engine) is an AI-powered automated advertising management platform. Users interact with a unified conversational AI agent embedded in the web interface to manage all advertising operations through natural language.

## Core Concept

**One entry point, one conversation, multiple capabilities**

Users access the system through the Web Platform and complete all operations via an embedded AI Agent chat interface without switching between different tools.

## Key Features

- **Conversational AI Interface**: Embedded chat window (bottom-right) for all user interactions
- **Ad Creative Generation**: AI-powered creative generation and analysis
- **Performance Analytics**: Real-time reporting with AI-driven insights and anomaly detection
- **Landing Page Builder**: Automated landing page generation with A/B testing
- **Campaign Automation**: Intelligent ad creation and budget optimization
- **Market Intelligence**: Competitor analysis and trend insights

## Supported Platforms

- Meta (Facebook/Instagram) Ads
- TikTok Ads
- Google Ads (planned)

## User Flow

1. User logs into Web Platform dashboard
2. Clicks AI chat icon (bottom-right corner)
3. Interacts with AI agent using natural language
4. AI agent coordinates multiple capability modules to fulfill requests
5. Results displayed in chat with embedded charts/cards
6. User can also view detailed data in dedicated management pages
